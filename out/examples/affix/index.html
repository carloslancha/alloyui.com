Getting Started

First load the seed and CSS files, if you haven't yet.

<script src="http://cdn.alloyui.com/3.0.0pr1/aui/aui-min.js"></script>
<link href="http://cdn.alloyui.com/3.0.0pr1/aui-css/css/bootstrap.min.css" rel="stylesheet"></link>

Then initialize AlloyUI and load the Ace Editor module.

YUI().use(
  'aui-affix',
  function(Y) {
    // code goes here
  }
);
For more information about configuration, check out our API Docs.
